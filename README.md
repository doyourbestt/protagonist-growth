# 主角成长手册

带轻游戏感的自我成长体验产品

## 🌊 项目介绍

**主角成长手册** 是一个面向年轻人的成长体验产品：

- 以书籍知识为成长骨架
- 结合动漫与电影式的画面感
- 用治愈、细腻、带一点梦幻轻奇幻的方式
- 陪伴你一点点成长

## 🎨 设计风格

- **主色调：** 雾蓝 (#7B9EAE)
- **风格：** 治愈 + 少年少女成长感
- **内容比例：** 内容体验 60%，互动实践 40%

## 🛠️ 技术栈

- 纯静态页面 (HTML/CSS/JS)
- GitHub Pages 托管
- 本地存储 (LocalStorage) 保存用户进度

## 📦 本地开发

```bash
cd protagonist-growth
python3 -m http.server 8080
```

然后访问 http://localhost:8080

## 🚀 自动部署

使用推送脚本：

```bash
./push.sh "你的更新说明"
```
